CSS перый видео-курс

1. Способы подключения CSS стилей

  1. В первом случае мы прописываем style, как атрибут к тегу p, данный способ считается не очень хорошим, и подходит только в том случае, если к объекту необходимо применить лишь несколько стилей.
    Данный способ является лучшем, чем 1, однако не идеальным, так как и html и CSS находятся в одном файле.
  2. Во втором случае, мы добавляем атрибут style в шапку - head и добавляем туда: "тег { прописываем стили сюда }"
  3. Мы прописываем в шапке html тег - link, с атрибутами rel="stylesheet" href="адресс CSS стилей", в которой уже и хранятся все стли. Самый правильный способ.

2. Селекторы выбора элемента

Селектор - элемент определяющий область действия стилей CSS. Это могут быть все теги HTML.

    Селекторы выбора:
  • Тег - можно указать тег, к которому будут применятся стили
  • Классы - class - мы прописываем его в виде атрибута и присваеваем ему определенное значение, в CSS мы указываем .имя класса к которым будут применятся стили. Один и тот же класс может быть внедрен во все необходимые теги.
  • Идентификатор - id - в отличие от class он может быть только один на всей странице. Каждому элементу можно присвоить свой идентификатор с уникальным именем, однако оно не должно повторятся. Это позволяет делать более точечную стилизацию. Для работы с id в CSS необходимо обращатся к ним начиная с #имя id.
  • Вложенные селекторы выбора - div p strong - это значит, что ко всем тегам - strong, которые находятся в абзацах p блоках, которые в свою очередь находятся в блоке div, будет применен стиль, который мы зададим. Обращатся к ним в CSS необходимо по такому же принципу.
  • Множественный выбор - div p strong, div em - через запятую можно перечислить все селекторы к которым мы хотим применить тот или иной стиль, будь то .class, #id или любой другой тег.

    Свойства:
  • text-decoration - варианты подчеркивания текста:
    • line-through - перечеркнутый текст
    • none - без подчеркивания
    • overline - подчеркнутый над текстом
    • underline - подчеркнутый под текстом
  • text-decoration-color - выбор цвета линии
  • cursor - выбор курсоров
  • focus - срабатывает пока курсор находится в поле
  • color - можно задавать цвета
  • content - указыываем, какой текст будет выводится до или после объекта, обычно используется с псевдоклассом after или before
    Псевдоклассы:
  • hover - стили будут применятся при наведении на объект
  • active - срабатывает при нажимании на что-либо
  • visited - псевдокласс применяемый в основном к ссылкам, по которым уже мы переходили. В основном прописывается только цвет
  • disabled - применяет стили к блокам, которые имеют атрибут disabled, то есть все поля, которые не активн
  • enabled - работает аналогично с disabled, но со всеми активными полями
  • required - работает аналогично с двумя предыдущими, пометка - обязательно к заполнению
  • empty - пустые абзатцы
  • not - некое усоловие, которое необходимо дописать в ()
    Псевдоэлементы - элементы, с помощью которых мы можем работать с каким-то определенным текстом элемента внутри html обьекта. Они прописываются с помощью ::
  • after - выводит некий текст, который будет находится сразу после объекта, его нельзя выделить
    • attr ( ) - позволяет выводить значения атрибутов
  • before - работает аналогично с after, но выводит текст до обьекта
  • first-letter/line - позволяет взаимодействовать с первым элементом/строкой блока
  • first/last-of-type - работает с первым/последним элементом данного типа

4. Работа с фоновыми изображениями

    Свойства:
  • background-color - с его помощью мы можем задавать цвет заднего фона
  • background-image - пзволяет воспользоваться методом url() и установить фоновую картинку
  • background-repeat - устанавливаем повторение картинки
  • background-position - указываем позицию картинки
    • Что бы не прописывать все эти свойства, можно сократить запись до background и подставлять через пробел значения предидущих параметров
